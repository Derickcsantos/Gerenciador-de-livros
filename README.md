# Gerenciador de livros
Projeto Fullstack desenvolvido com ASP.NET Core e arquitetura MVC, com intuito de desenvolver habilidades com C#

## Funcionalidades
- Adicionar livros
- Editar livros
- Visualizar livros em tabela
- Excluir livro

## Tecnologias
- ASP.NET
- HTML
- Bootstrap
- Git
